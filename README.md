# 📦 Gerenciador de Estoque

**Gerenciador de Estoque** é um sistema desktop desenvolvido para facilitar o controle de produtos, a visualização de itens cadastrados e a organização das informações de estoque em um único ambiente.

Com uma interface prática e recursos voltados para o dia a dia, o sistema permite gerenciar produtos, acessar configurações e manter o controle do estoque de forma mais simples.

---

## ✨ Funcionalidades

- **Login de Acesso**  
  Permite a autenticação do usuário antes do uso do sistema.

- **Cadastro de Produtos**  
  Realize o registro de novos produtos com as informações necessárias para controle.

- **Controle de Estoque**  
  Visualize os itens cadastrados e acompanhe a movimentação do estoque.

- **Configurações do Sistema**  
  Área destinada aos ajustes gerais da aplicação.

- **Configuração de Banco de Dados**  
  Permite definir e ajustar a conexão utilizada pelo sistema.

## 📁 Estrutura do Projeto

| Pasta/Arquivo | Descrição |
|--------------|-----------|
| `Data/` | Camada de acesso e manipulação de dados |
| `Infra/` | Configurações e infraestrutura do sistema |
| `Models/` | Modelos de entidades da aplicação |
| `Services/` | Regras de negócio e serviços |
| `Telas/` | Interfaces e formulários do sistema |
| `Utils/` | Funções auxiliares e utilitárias |
