TCC - Gerenciador de Estoque
Sistema de gerenciamento de estoque desenvolvido para controle de produtos, autenticação de usuários e configurações do sistema, com interface desktop e organização por telas para facilitar o uso e a administração.
Funcionalidades
Login de acesso ao sistema
Cadastro de produtos
Controle e visualização de estoque
Tela de configurações gerais
Configuração de banco de dados
Tecnologias utilizadas
C#
.NET
Windows Forms / Desktop App
Banco de Dados
Estrutura do projeto
O projeto foi organizado em camadas e pastas para separar responsabilidades, facilitando manutenção e evolução do sistema.

Objetivo do projeto
Este sistema foi desenvolvido com o objetivo de facilitar o gerenciamento de estoque, permitindo melhor controle de produtos, organização das informações e praticidade no uso diário.
Como executar
Clone este repositório:
```bash
   git clone https://github.com/sr4kkk/Tcc-gerenciador-de-estoque.git
   ```
Acesse a pasta do projeto:
```bash
   cd Tcc-gerenciador-de-estoque
   ```
Abra o projeto no Visual Studio ou Visual Studio Code.
Restaure as dependências e execute o projeto.
